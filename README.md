# ⚡ Hogar & Estilo ZB — Sitio Web (React + Vite)

Versión evolucionada del sitio web de la tienda **Hogar & Estilo ZB**, desarrollada como aplicación de página única (SPA) con React y Vite.

## 📋 Descripción

Segunda versión del sitio web de Hogar & Estilo ZB, migrada a React para una experiencia más dinámica e interactiva. Mantiene el diseño de la versión HTML/CSS pero con arquitectura de componentes, enrutamiento con React Router y mejor rendimiento gracias a Vite como bundler.

---

## ✨ Funcionalidades

- Navegación entre páginas sin recarga (SPA con React Router)
- Catálogo de productos por categoría (baño, cocina, dormitorio, sala, salud, uso personal)
- Banner dinámico con imágenes promocionales
- Sección de beneficios, ventajas y comentarios
- Marquee de ofertas animado
- Temporizador de cuenta regresiva
- Formas de pago y contacto
- Footer con redes sociales

## 📁 Estructura del proyecto

```
HOGAR_ESTILO_ZB_JSX/
├── src/
│   ├── components/    # Banner, Navbar, Footer, Beneficios, Ofertas...
│   ├── pages/         # Inicio, Tienda, Nosotros, Contacto, FormasPago
│   ├── styles/        # CSS por página
│   ├── data/          # datos.js (catálogo de productos)
│   ├── assets/        # Imágenes y recursos
│   ├── App.jsx        # Rutas principales
│   └── main.jsx       # Punto de entrada
├── public/
│   └── productos/     # Imágenes de productos por categoría
├── index.html
├── vite.config.js
└── package.json
```
